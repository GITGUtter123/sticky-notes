Instructions
Note Inventory

This extension allows you to add sticky notes to any web page.

How to Use:

  • Click the "+" button on any webpage to add a new note
  • Drag the note header to move it around
  • Click the "Ã—" to hide a note (it will be saved in inventory)
  • Change note color by clicking the color dots in the header
  • Notes are automatically saved per webpage
  • Import/Export your notes using the buttons below

Your hidden notes are saved here. You can restore or permanently delete them.
